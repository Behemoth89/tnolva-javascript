function onCameraError(err: EmittedError) {
  switch (err.name) {
    case 'NotAllowedError': toast.error('Camera permission denied') break
    case 'NotFoundError': toast.error('No camera found') break
    case 'InsecureContextError': toast.error('HTTPS required for camera') break
    case 'StreamApiNotSupportedError': toast.error('Camera not supported in this browser') break
    default: toast.error(`Camera error: ${err.message}`)
  }
}

**Props:**
- `contestId: string` — contest UUID for routing
- `userTeamId: string` — user's team registration UUID (from UserTeamListItem.id)

**Emits:**
- `scan-success(checkPointId: string, displayCPId: string)` — when a marking is successfully submitted
- `scan-error(message: string)` — when an error occurs
- `camera-error(message: string)` — when camera fails

**Internal state:**
- `pause: ref&lt;boolean&gt;` — controls QrcodeStream paused state
- `manualInput: ref&lt;string&gt;` — bound to manual entry text input
- `isSubmitting: ref&lt;boolean&gt;` — prevents double-submission

**Camera scanning (QrcodeStream):**
- Import `QrcodeStream` from `vue-qrcode-reader`
- Import `useRaceStore` and `useToast`
- Import `he` from `he` for HTML entity decoding
- On `@detect`: extract `detectedCodes[0].rawValue`, decode with `he.decode()`, call `handleCPId(rawValue)`
- On `@error`: emit `camera-error` with human-readable message based on error name
- Bind `:paused="pause"` and `:constraints="{ facingMode: 'environment' }"`
- Use a `pause` ref: set true while submission in-flight, reset after 2000ms timeout
- Use `facingMode: 'environment'` for rear camera (D-02: camera auto-shows when modal opens — no separate tap-to-start)

**handleCPId(cpId: string) function:**
1. Decode the cpId with `he.decode(cpId)` for display
2. Call `raceStore.submitScan(cpId, props.userTeamId)`
3. Based on result:
   - If `isAlreadyScanned`: call `toast.info('Already scanned')` — NO error state surfaced to user (D-15)
   - If `!statusOk`: call `toast.error(message || 'Scan failed')` and emit `scan-error`
   - If `statusOk && !isAlreadyScanned`: call `toast.success(...)` and emit `scan-success`
4. Set `pause = true`, setTimeout 2000ms to reset `pause = false`

**Manual entry:**
- Text input with `v-model="manualInput"`
- Placeholder: "e.g. OPEN-CP-1" (D-07)
- On submit button click (NOT on Enter keypress per D-06): call `handleManualSubmit()`
- `handleManualSubmit()`: trim input, convert to uppercase, validate non-empty, call `handleCPId(uppercased)`
- Clear `manualInput` after successful submit

**Template structure:**
- Outer div with scanner styles
- `QrcodeStream` component with camera preview
- Manual entry section below camera: text input + submit button
- Optional: show a small "scanning..." overlay when `pause === true`

**CSS:**

- Manual entry section below the camera with clear visual separation
