<task type="auto">
  <name>Task 1: Create multi-stage Dockerfile</name>
  <files>Dockerfile</files>
  <read_first>
    - package.json — build script is `tsc -b && vite build`, project name is `react-todo`
    - vite.config.ts — build output defaults to `dist/` directory
    - .planning/phases/05-docker-deployment/05-CONTEXT.md — locked decisions D-01 through D-05
  </read_first>
  <action>
    Create a multi-stage Dockerfile with two stages:

    **Stage 1 — Build (node:20-alpine):**
    - FROM node:20-alpine AS build
    - WORKDIR /app
    - Copy package.json and package-lock.json first (layer caching)
    - RUN npm ci (production + dev deps needed for build)
    - Copy all source files (COPY . .)
    - ARG VITE_API_URL with default value "https://taltech.akaver.com"
    - ENV VITE_API_URL=$VITE_API_URL (Vite reads env vars at build time)
    - RUN npm run build (produces dist/ directory)

    **Stage 2 — Serve (nginx:stable-alpine):**
    - FROM nginx:stable-alpine AS serve
    - Copy custom nginx config: COPY nginx.conf /etc/nginx/conf.d/default.conf
    - Copy built assets from build stage: COPY --from=build /app/dist /usr/share/nginx/html
    - EXPOSE 80
    - CMD ["nginx", "-g", "daemon off;"]

    Key requirements per locked decisions:
    - D-02: Must use node:20-alpine for build, nginx:stable-alpine for serve
    - D-05: VITE_API_URL must be a build ARG (not runtime env)
    - D-04: No reverse proxy — SPA connects directly to API via CORS
  </action>
  <acceptance_criteria>
    - Dockerfile exists in project root
    - Contains "FROM node:20-alpine" (build stage)
    - Contains "FROM nginx:stable-alpine" (serve stage)
    - Contains "ARG VITE_API_URL" with default value
    - Contains "ENV VITE_API_URL" (passes ARG to build env)
    - Contains "npm run build" in build stage
    - Contains "COPY --from=build" in serve stage
    - Contains "COPY nginx.conf /etc/nginx/conf.d/default.conf"
    - Contains "EXPOSE 80"
    - Contains exactly 2 FROM statements (multi-stage)
  </acceptance_criteria>
  <verify>
    <automated>test -f Dockerfile && grep -c "FROM" Dockerfile | grep -q "2" && grep -q "ARG VITE_API_URL" Dockerfile && grep -q "nginx:stable-alpine" Dockerfile && echo "PASS" || echo "FAIL"</automated>
  </verify>
  <done>Dockerfile exists with multi-stage build (node:20-alpine → nginx:stable-alpine), VITE_API_URL build ARG, and nginx.conf reference</done>
</task>
