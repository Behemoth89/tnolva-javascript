       - Since we only have the single priority object (not the full sorted list), use a simpler approach:
       - Pass all priorities to TaskCard (already done via props), find the sorted order, then map the current priority's index to a color
       - Implementation:
         ```typescript
         const getPriorityColor = (priority: Priority, allPriorities: Priority[]): string => {
           const sorted = [...allPriorities].sort((a, b) => a.prioritySort - b.prioritySort);
           const index = sorted.findIndex(p => p.id === priority.id);
           if (index === 0) return 'text-red-500';       // lowest sort = urgent
           if (index === sorted.length - 1) return 'text-green-500'; // highest sort = low
           return 'text-amber-500'; // middle
         };
         ```
